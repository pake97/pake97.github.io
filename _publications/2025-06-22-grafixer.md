**Grafixer** is an interactive tool for human-in-the-loop repair of property graphs. Users upload property graph datasets, define constraints using **Cypher**, and collaboratively identify and correct inconsistencies. The system manages overlapping violations so multiple users can work simultaneously without conflicts, while an interactive dashboard lets administrators monitor real-time statistics, track progress, and oversee the workflow.
